Music
Command Arguments Description
Play {song} Searches YouTube/Soundclouse for the song entered
Queue None Shows songs in the queue
Skip None Skips the song currently playing.
Playing None Shows progress and name fo the song currently playing.
Art None Gets the art/thumbnail of the song currently playing.
Leave None Causes Tsubasa to stop playing the current song and leave.
Join None Causes Tsubasa to join the voice channel of the command user.
Volume {0-150} Sets the volume of the music playing to the selected volume
Pause None Pauses the currently playing track.
Resumes None Resumes the track if it was paused.
Seek {time(seconds)} Seeks to the time in the song supplied.
Lyrics None NOTE: Buggy - Gets lyrics for the current song if supported.
